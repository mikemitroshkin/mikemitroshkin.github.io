Классический дизайн игрового поля для покера

UI design © Mike Mitroshkin

Целью этого проекта является оформление игрового поля для покера в классическом стиле. Задачей заказчиков включает в себя создание несложного управления игровым процессом и взаимодействия пользователей при узнаваемом интерфейсе для целевой аудитории, а также для начинающих игроков.
